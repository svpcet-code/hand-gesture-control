Control your system and media using hand gestures via webcam.

## Features
- Play / Pause media
- Next / Previous track
- Mute / Unmute
- Close active application
- Real-time hand detection

## Tech Stack
- Python 3.10
- OpenCV
- MediaPipe
- PyAutoGUI

## Controls
| Fingers | Action |
|-------|--------|
| 5 | Play / Pause |
| 4 | Next Track |
| 3 | Previous Track |
| 2 | Mute |
| 1 | Close App |
